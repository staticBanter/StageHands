Root Elements

For some simplicity StageHands classifies both the HTML and the BODY tags as "Root Elements", while in reality they are actually "Main Root" and "Sectioning Root" respectively.

HTML(Main Root) Element.

StageHands will apply the following styles to all HTML elements.

min-width: $screenWidthMin;
max-width: $screenWidthMax;
min-height: $screenHeightMin;
max-height: $screenHeightMax;
height: 100%;

Body(Sectioning Root) Element

StageHands will apply the following styles to all BODY elements.

background-color: $white;
height: 100%;
