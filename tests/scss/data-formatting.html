Data Formatting

StageHands formats certain pieces of data in specific ways in order to have a uniform structure. Because of this we will list any data structures that StageHands has so you may extend to them or create your own.

Log Message Data

List

$messageLog_List:(
    (
    "CODE"
        (
            ("SUB_CODE" "MESSAGE"),
            ("SUB_CODE" "MESSAGE")
            ...
        )
    ),
    (
    "CODE"
        (
            ("SUB_CODE" "MESSAGE"),
            ("SUB_CODE" "MESSAGE")
        ...
        )
    ),
...
)

Map 

$messageLog_Map:(
"CODE":
    (
        "SUB_CODE":"MESSAGE",
        "SUB_CODE":"MESSAGE"
        ...
    ),
"CODE":
    (
        "SUB_CODE":"MESSAGE",
        "SUB_CODE":"MESSAGE"
        ...
    ),
...
)
