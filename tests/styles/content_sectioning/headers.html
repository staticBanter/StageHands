Headers

StageHands will apply the following styles to all HEADER elements.

StageHands currently does not apply any styles to this element by default.

Main Header

This is usually the first header of the document body displayed at the top of the page. This header usually contains the applications name,

and some navigational elements to navigate the entire application.

StageHands will apply the following styles to all 'Main Header' elements.

background-color: $primary;
color: $white;
min-height: 10%;
height: 10%;
max-height: max-content;
overflow: hidden;
