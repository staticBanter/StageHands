Forms

StageHands will apply the following styles to all FORM elements.

font-family: globals.$neutralFontFamily;
color: globals.$neutralTextColour;
line-height: globals.$neutralTextLine;
font-size: globals.$neutralTextHeight;
padding: globals.$padding

Form Separators.

Form Separatorsare one of the following elements that is a direct child of aFORM

  • LABEL
  • SECTION
  • DIV

Please note that when using a Generic Separator Forms, it is recommended that you be consitant with the elements that you chose to be the separator and which ones are nested separators.For example;

form
    section
        label
        input
    section
        div
            label
            input
        div
            label
            input
    section
        div
            label
            input
In this example the SECTION element was the separator and theDIV was the nested. Of course you could have the SECTION be nested and DIV the separator.

Try to avoid doing this;

form
    section
        label
        input
    div
        section
            label
            input
        div
            label
            input
    section
        div
            label
            input
where the separating element is being changed.

While there is no proper way to structure a form, StageHands does prefer and recommend using Label Separated Forms, as it is less HTML, more consistent and more accessible

While we try to create a unified design no matter the structure choice you pick below, there are some minor discrepancies between the two.

Multiple Direct Children

The following styles will be applied to any element that is a direct sibling of aFORMand preceeds another direct sibling.

margin: globals.$margin-large auto;
