MENUS

StageHands will apply the following styles to all MENUS elements.

padding-left:unset;

This element is still considered experimental technology and is not supported by many browsers.Also at this time StageHands only supports the toolbars type. Other types of menus have not been tested with this project yet.

StageHands discourages the use of the MENUITEM as it is currently not a standard HTML element, see; Mozilla Developer | Obsolete and Deprecated Elements. Instead please use the BUTTON element.

Menu List Styles

The following styles will be applied to any element that is a direct child of any <LI>element that is a child of a <MENU> or any child <MENU>

padding-left: globals.$padding-large;
