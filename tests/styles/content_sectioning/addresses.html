Addresses

StageHands will apply the following styles to all ADDRESS elements.

display: inline-block;
width: max-content;
padding: $padding;
box-sizing: border-box;
font-family: $bodyFontFamily;

Addresses that are beside one another;

margin: auto $margin;

Direct descendent of the address element;

margin: auto $margin;

Address Lists

It is sometimes common to place all of the address data in a list style format.For this StageHands will apply the following styles to all ul/ol elements,

list-style-type: none;
margin: 0;
padding: 0;

List Elements of Address List

Every LI element in the address list.

margin: auto $margin;

Address Lists example; Note The styling looks identical to the default address element styles.
