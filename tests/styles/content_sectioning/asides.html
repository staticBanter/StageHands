Asides

StageHands will apply the following styles to all ASIDE elements.

StageHands currently does not apply any styles to this element by default.

Complementary Aside

This is the usually the first aside of the body that is either before or after the main element.

Note StageHands currently only supports asides that are placed before the main element 

width: 20%;
display: inline-block;
padding: $padding;
box-sizing: border-box;
background-color: $secondary;
border-radius: 0 10px 10px 0;

Remaining elements

The remaining children of the complementary aside will be styled like so;

color: $white;
border: $border;
border-radius: $border-rounded;
padding: $padding;
