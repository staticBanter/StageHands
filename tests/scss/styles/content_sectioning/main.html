Main

StageHands will apply the following styles to all MAIN elements.

The first element related to the body with the role main have the following styles applied.

overflow: auto;
min-height: 80%;
max-height: max-content;
height: 100%;
padding: $padding;
box-sizing: border-box;

Mains with Complementary Asides

If the main is preceded by an aside element it will be styled like so;

float: right;
width: 80%;
