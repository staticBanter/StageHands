Global Variables

These are variables that can be accessed by any stylesheet and are used on every page. All global variables can also be accessed using the CSS var(--varName) method.

Screen Sizes.

StageHands will set a global minimum and maximum values for the screens height and width, these values are called;

$screenWidthMin: 240px !default;
$screenWidthMax: 3840px !default;
$screenHeightMin: 320px !default;
$screenHeightMax: 2160px !default;

They will restrict the minium and maximum sizes of the browser window.

Colour Theming.

Element Colours

These are colours that would be applied to elements and element decorations, these styles could be applied on properties like;background-color, border-color, box-shadow-color, etc... (Though in reality you can use these variables for anything you choose.)

$danger: rgba(255,0,0,1) !default;
$success: rgba(0,255,0,1) !default;
$white: rgba(246,246,246,1) !default;
$black: rgba(0,0,0,1) !default;
$primary: rgb(92,92,92,1) !default;
$secondary: color.complement($primary);
$accent: color.invert($secondary);

These colours are gray-scale on purpose, though boring, this is to encourage the creation of a custom colour theme.

Text Colours

These are colours that are used on text.(ie, they should be used on the color property.)

$neutralTextColour: rgba(0,0,0,1) !default;
$activeTextColour: $accent !default;
$hoverTextColour: $secondary !default;
$visitedTextColour: $accent;
$linkedTextColour: $neutralTextColour;

Borders 

$border-colour: $accent !default;
$border-rounded: 10px !default;
$border-thickness: 3px !default;

$border: $border-thickness solid $border-colour !default;
$border-grooved: $border-thickness grooved $border-colour !default;

Box Shadows 

$boxShadow-right: 0px !default;
$boxShadow-bottom: 0px !default;
$boxShadow-radius: 20px !default;
$boxShadow-blur: -($boxShadow-radius / 2) !default;
$boxShadow-colour: rgba(0,0,0,1) !default;
$boxShadow: $boxShadow-right $boxShadow-bottom $boxShadow-radius $boxShadow-blur $boxShadow-colour !default;

Margin & Padding.

$padding: 0.5rem !default;
$padding-large: ($padding * 2) !default;
$padding-xLarge: ($padding * 3) !default;
$padding-small: ($padding / 2) !default;

$margin: 0.5rem !default;
$margin-large: ($margin * 2) !default;
$margin-xLarge: ($margin * 3) !default;
$margin-small: ($margin / 2) !default;

Font Size and Line Heights.

$bodyTextHeight: 1rem;
$bodyTextLine: 1.2rem;
$headerTextHeight: 2rem;
$headerTextLine: 2.6rem;
$neutralTextHeight: 1.4rem;
$neutralTextLine: 1.6rem;
$displayTextHeight: 2rem !default;
$displayTextLine: 2.6rem !default;

Font Families

StageHands breaks fonts into four different categories;

  • Header-Fonts : Uses a Script/Serif typeface,
  • Body-Fonts : Uses a Sans Serif typeface,
  • Neutral-Fonts : Uses a Sans Serif typeface,
  • Display-Fonts : Uses a Script typeface.
// Header And Title Fonts
$headerFontFamily: Dancing_Script, "Lucida Handwriting", "Times New Roman", Times;

// Body Text Fonts(TODO).
$bodyFontFamily: JetBrains_Mono, "Times New Roman", "Times";

// Neutral Text Fonts(TODO).
$neutralFontFamily: Inconsolata, "Sans Serif";

// Display Font Text.
$displayFontFamily: Sansita_Swashed;

StageHands is also preparing to support *Variable Fonts*. This option is enabled by the ```$variableFonts``` global variable. Though this feature isn't fully supported/tested.

Overriding Variables

You may override any global variable by using its name and providing a new value

Example;

//customStyles.scss

//Importing StageHands.
@use "StageHands";

$screenWidthMin: 480px; // Override for customStyles.scss.
