Asides

StageHands will apply the following styles to all ASIDE elements.

font-family: globals.$bodyFontFamily;
color: globals.$neutralTextColour;
line-height: globals.$bodyTextLine;
font-size: globals.$bodyTextHeight;

Complementary Aside

This is the usually the first aside of the body that is either before or after the main element.

An example of a Complementary Aside would be the one you currently see on the right side of the page.

Note StageHands currently only supports asides that are placed before the main element 

width: 20%;
display: inline-block;
padding: globals.$padding;
box-sizing: border-box;
background-color: globals.$secondary;
border-radius: 0 10px 10px 0;

Remaining Elements

The remaining children of the complementary aside will be styled like so;

color: globals.$white;
padding: globals.$padding;
